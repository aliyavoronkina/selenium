# Автоматизация тестирования формы заказа карты

## Описание проекта
Проект содержит автоматизированные тесты для проверки функциональности формы заказа банковской карты.

## Технологии
- Java 17
- JUnit 5
- Selenide
- Gradle
- GitHub Actions
